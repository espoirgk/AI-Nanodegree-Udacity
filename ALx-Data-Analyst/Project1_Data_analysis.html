Introduction¶

Dataset Description¶

The environmental data used in this project comes from Gapminder.\ The platform Gapminder has collected a lot of information about how people live their lives in different countries, tracked across the years, and on a number of different indicators. We have selected 4 variables of interest which are time series for our study.

CO2 emission\ This data represents Carbon dioxide emissions from the burning of fossil fuels (Unit: metric tonnes of CO2 per personn).

Forest area\ This dataset represent land under natural or planted stands of trees of at least 5 meters in situ, whether productive or not, and excludes tree stands in agricultural production systems (Unit: sq.km).

Natural gas production \ Conventional Natural Gas Production Natural gas is primarily extracted from gas and oil wells. (Unit: tons oil equivalent).

Sustainable Development Index \ The Sustainable Development Index is an efficiency metric, designed to assess the ecological efficiency of nations in delivering human development.\ It is calculated as the quotient of two figures:

  • a development index based on the Human Development Index, calculated as the geometric mean of the life expectancy index, the education index, and a modified income index;
  • an ecological impact index calculated as the extent to which consumption-based CO2 emissions and material footprint exceed per-capita shares of planetary boundaries.

Questions¶

  1. What are the average dynamics of CO2 emissions, the state of forest cover, natural gas, and the Sustainable Development Index?\ 
     a) What can we learn from the dynamics of these variables at the global level?\
 b) Is there a relation between forest decrease and natural gas production?
  2. Analyze the 10 countries which had the largest forest cover and the highest natural gas production in 2018.\ a) In 2018, which 10 countries had the largest forest cover and the highest natural gas production?\ b) Which countries are on both lists of largest forest cover and the highest natural gas production in 2018?
  3. Natural gas production by African countries.\ a) How many and which African countries have produced natural gas?\ b) What is the dynamics of their production? Which country produces the most? Which is the lowest producer?
  4. The Sustainable Development Index (SDI) measures the ecological efficiency of nations in terms of human development.\ a) Is there a link between each of these 3 variables and the sustainable development index?

## Function to get dataframe's shape
def shape_function(list_dataframe, name_dataframe):
    """
    This function print the number of columns and rows of your datasets

    Parameters:
    list_dataframe: list of dataframe you have created.
    name_dataframe: list of your dataframe's name
    
    """
    if not (isinstance(list_dataframe, list) and isinstance(name_dataframe, list)):
        print("ERROR: All your Parameters must be a list.")
    else:
        if len(list_dataframe) != len(name_dataframe):
            print("ERROR: Length of Parameters must be same.")
        else:
            i = 0
            for ds in list_dataframe:
                print(f"The {name_dataframe[i]} dataset contains {ds.shape[1]} columns and {ds.shape[0]} rows.")
                i +=1

def melt_function(data, name_value):
    
    """
    This function print the number of columns and rows of your datasets

    Parameters:
    data: dataframe to melt
    name_value: customize value output
    
    """
    ls_cols = data.columns.to_list()[1:]
    
    return data.melt(id_vars=['country'], value_vars=ls_cols, var_name='year', value_name=name_value)

def decode_rows(data, col):
    
    """
    This function extrat and convert each row from object dtype to numerical by decoding:
    code: B, M, K respectively for Billion, Million and Thousand 
    e.g : 81.2M = 81.2x10E6, 81.2B = 81.2x10E9

    Parameters:
    data: dataframe 
    col: column of interest
    
    """
    if data[col].dtypes == 'float' or data[col].dtypes == 'int':
        
        #print(f"You don't need to decode this columns. The type of {col} is {data[col].dtypes}.")
        
        return data

    else:
        
        
        extract_data = data[col].str.extract(r'(?P<digit>[\d\.\d]+)')
        extract_data["letter"] = data[col].str.extract('([a-zA-Z ]+)')

        if extract_data.digit.dtypes != 'float':
            extract_data.digit = extract_data.digit.astype(float)

        extract_data.letter = extract_data.letter.apply(lambda x: 1000000000 if str(x).startswith('B') \
                                                        else (1000000 if str(x).startswith('M') \
                                                              else (1000 if str(x).startswith('k') else 1)))

        data[col] = extract_data["digit"]*extract_data["letter"]
    
        return data

# Skewness and kurtosis stats
def distribution_stats(data, columns):
    from scipy import stats
    
    #loop to calculate stats parameters
    for col in columns:
        sk = stats.skew(data[col].dropna(), bias=False)
        kurt = stats.kurtosis(data[col].dropna(), bias=False)
        #Check and print conclusion depend on stats parameters value
        
        skew_comparison = lambda x : "asymmetrical to the right" if  x > 0 else ("normal and symmetric" 
                                                                                 if x==0 else "asymmetrical to the left")

        kurtosis_comparison = lambda y : "sharp (leptokurtotic)" if  y > 3 else ("normal" 
                                                                                 if y==3 else "crushed (playkurtotic)")
    
        print(f"The skew value of {col} is equal to {round(sk, 2)}. The distribuion is therefore {skew_comparison(sk)}.")
        print(f"The kurtosis value of {col} is equal to {round(kurt, 2)}. The distribuion is therefore {kurtosis_comparison(kurt)}.")
        print("")
        print("=======================================================================================================")

Conclusions¶

In this analysis, we notice a clear decrease in the world's forests in the last few years. In contrast, gas production continues to increase.\ By definition the SDI is related to the environmental concept, we did not observe any link between the SDI and the other environmental parameters.

Limitation¶

For other perspectives, it would be interesting to add data on populations, life expectancy, and GDP to better understand these relationships with sustainable developpement index. This addition of more data will make some of the previous results more meaningful.
